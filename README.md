# Lista obecności 3

Program konsolowy w C++ do zarządzania listami obecności.

# Opis

Program umożliwia:
- dodawanie osób do bazy
- tworzenie kilku list obecności
- przypisywanie osób do wybranej listy
- ustawianie obecności
- usuwanie osoby z listy
- zmianę danych osoby
- wyświetlanie wszystkich list w formie tabeli

W tej wersji program jest podzielony na klasy:
- `Osoba` — przechowuje dane osoby,
- `ListaObecnosci` — obsługuje jedną listę obecności,
- `InterfejsUzytkownika` — odpowiada za menu i komunikację z użytkownikiem.
