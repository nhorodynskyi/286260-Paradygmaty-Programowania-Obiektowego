# Opis projektu
Program w języku C++ do obslugi listy obecności studentów i pracowników.

# Końcowy program umożliwia:
- dodawanie studentów/pracowników
- dodawanie list obecności
- dodawanie studentów/pracowników do list obecności
- ustawienie obecności
- usunienie osób z list obecności
- zmiana danych osób
- wyświetlanie list obecności
- zapisywanie list obecności do plików (txt/csv)
