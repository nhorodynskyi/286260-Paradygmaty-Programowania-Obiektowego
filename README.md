# Lista obecności 1
Prosty program konsolowy w C++ do obsługi listy obecności studentów.

#  Opis 

Program umożliwia:
- dodawanie studentów
- zmianę obecności
- wyświetlanie listy
- edycję danych studenta
- usuwanie studenta z listy

Dane studentów są przechowywane w dynamicznych tablicach.
