# Lista obecności 2

Program konsolowy w C++ do obsługi listy obecności studentów.

# Opis

Program pozwala:
- dodać osobę do listy
- ustawić obecność
- wyświetlić osoby obecne
- wyświetlić osoby nieobecne
- edytować dane osoby
- usunąć osobę z listy

W tej wersji dane są zapisane w klasie `Osoba`, a lista jest obsługiwana przez klasę `listaObecnosci`.
