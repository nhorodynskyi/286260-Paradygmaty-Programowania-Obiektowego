# Lista obecności 5

Program konsolowy w C++ do zarządzania listami obecności studentów i pracowników.

# Opis

Program umożliwia:
- dodawanie studentów i pracowników do bazy
- tworzenie list obecności
- przypisywanie osób do wybranej listy
- ustawianie obecności
- usuwanie osoby z listy
- edycję danych osoby
- wyświetlanie list w formie tabeli

W tej wersji dodano interfejs `Serializable`, który pozwala serializować obiekty do tekstu.
- `Serializable` — interfejs do zamiany obiektu na tekst.
